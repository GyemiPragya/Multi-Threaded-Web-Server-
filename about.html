About This HTTP Server Project

This project demonstrates a multi-threaded HTTP server written entirely in modern C++.
It can serve static HTML files and is extensible for learning OS/network principles!

Why C++? Main Features:

FeatureBenefit
ThreadingHandles multiple clients at once
Low-level socketsDirect control for learning OS internals
CustomizabilityEasy to add logging, caching, and security

This project was made for OS PBL
